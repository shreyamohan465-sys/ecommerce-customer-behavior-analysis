 🛒 Ecommerce Customer Behavior Analysis

📌 Project Overview
The Ecommerce Customer Behavior Analysis project focuses on analyzing customer shopping patterns and purchasing trends using data analytics techniques.  
The goal is to extract meaningful insights from ecommerce data to help businesses understand customer behavior and improve decision-making.

This project uses Python for data cleaning and analysis, SQL for querying customer data, and Power BI for creating interactive dashboards.



🎯 Objectives
- Analyze customer shopping behavior
- Identify popular product categories
- Understand purchasing trends
- Analyze payment methods used by customers
- Visualize insights using an interactive dashboard



 🛠 Tools and Technologies
- Python
- SQL
- Power BI
- Pandas
- Matplotlib / Seaborn
- CSV Dataset
